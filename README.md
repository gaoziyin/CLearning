# C语言快速学习教程

## 📚 课程结构

本教程将C语言知识点划分为8个主要章节，每个章节包含详细的示例代码和中文注释。

### 第一章：基础语法 (01_Basics)
- 1.1 Hello World - 第一个C程序
- 1.2 变量与数据类型 - 基本数据类型详解
- 1.3 常量 - 常量的定义与使用
- 1.4 运算符 - 算术、关系、逻辑运算符
- 1.5 类型转换 - 隐式与显式转换
- 1.6 输入输出 - printf和scanf详解

### 第二章：控制流程 (02_ControlFlow)
- 2.1 if语句 - 条件判断
- 2.2 switch语句 - 多分支选择
- 2.3 for循环 - 计数循环
- 2.4 while循环 - 条件循环
- 2.5 do-while循环 - 后测试循环
- 2.6 跳转语句 - break、continue、goto

### 第三章：函数 (03_Functions)
- 3.1 函数定义与调用 - 函数基础
- 3.2 函数参数 - 值传递
- 3.3 函数返回值 - return语句
- 3.4 递归函数 - 递归原理与应用
- 3.5 作用域 - 局部变量与全局变量
- 3.6 static和extern - 存储类别

### 第四章：数组与字符串 (04_ArraysStrings)
- 4.1 一维数组 - 数组基础
- 4.2 多维数组 - 二维数组应用
- 4.3 字符数组 - 字符串基础
- 4.4 字符串函数 - string.h库
- 4.5 数组作为函数参数 - 数组传递

### 第五章：指针 (05_Pointers)
- 5.1 指针基础 - 指针的概念
- 5.2 指针运算 - 指针的算术运算
- 5.3 指针与数组 - 数组指针
- 5.4 指针与字符串 - 字符串指针
- 5.5 指针与函数 - 函数指针
- 5.6 多级指针 - 指针的指针

### 第六章：结构体与联合体 (06_StructsUnions)
- 6.1 结构体定义 - struct关键字
- 6.2 结构体操作 - 访问成员
- 6.3 结构体数组 - 结构体集合
- 6.4 结构体指针 - 指向结构体的指针
- 6.5 联合体 - union关键字
- 6.6 枚举类型 - enum关键字
- 6.7 typedef - 类型定义

### 第七章：文件操作 (07_FileIO)
- 7.1 文件基础 - 文件指针
- 7.2 文本文件读写 - fprintf、fscanf
- 7.3 字符文件操作 - fgetc、fputc
- 7.4 行文件操作 - fgets、fputs
- 7.5 二进制文件 - fread、fwrite
- 7.6 文件定位 - fseek、ftell

### 第八章：高级主题 (08_Advanced)
- 8.1 动态内存分配 - malloc、free
- 8.2 预处理器 - #define、#include
- 8.3 宏定义 - 宏的高级用法
- 8.4 条件编译 - #ifdef、#ifndef
- 8.5 位运算 - 位操作符
- 8.6 常用标准库 - stdlib.h、math.h
- 8.7 命令行参数 - argc、argv

## 🚀 如何使用

1. 按照章节顺序学习
2. 阅读每个示例代码中的中文注释
3. 编译并运行示例程序
4. 修改代码，实践练习

## 🔨 编译说明

### 使用GCC编译器：
```bash
gcc filename.c -o filename.exe
```

### 运行程序：
```bash
./filename.exe
```

## 📝 学习建议

- 从第一章开始，循序渐进
- 每个示例都要亲自编译运行
- 尝试修改代码，观察结果变化
- 完成每章后的练习题
- 多写代码，多调试
